Project Description

Data Source

The data for this project come from the College Scorecard API, developed and hosted by the US Department of Education. This rich dataset contains thousands of variables on thousands of institutions, their students, and those students' outcomes. These data are collected through the annual IPEDS reports required of all institutions receiving Title IV aid funding for students (including Pell Grants, subsidized loans, and federal work-study).

API Call

The data were accessed through an API call programmed in Python using the json, requests, csv, and time packages. Due to the limitations of the College Scorecard API and the potential size of calls, two measures were implemented: the request was limited to one year (2013) and one state (California, with the largest community college system in the country) and the call was slowed with the sleep command. The data pulled were stored in a CSV file for analysis.

Analysis

Results were produced using the statsmodels.api module for Python. As discussed more in depth in the Methodology and Results sections, an ordinary least squares regression was run on a number of independent variables to determine their effect on graduation rates.
